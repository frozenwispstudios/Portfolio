Across Kingdoms (VR)

Protect the various kingdoms from the hordes of enemies - humans or monsters. Shoot them all before they rip down the gates and destroy the crystals that link the worlds. Across Kingdoms is an archery VR game built in 6 weeks by a team of 3 for a university project. For this project I handled programming the A.I, creating the level design and the U.I design. The 2 other members programmed the player, U.I, and worked on polishing the game adding explosive barrels and different particles. This project was a solid build that, with a little more time, could become a complete project ready for sale. This project was developed for the Windows Mixed Reality headset.

Shelf Generator Tool

This is a simple tool to help generate shelves in 3D games with bottles, books, weapons, and more. This is an in-editor tool that is easy to use and saves time placing meshes/prefabs for you. I built this while learning the Unity tool built library alongside a simple version of poly brush. I still need to upload this to the asset store which will require a video to be made on the tool.

Terrain Generator

This Terrain generator was created for my Computer Graphics class and shows building polygons live to generate terrain with a height texture map, and a simple random generation algorithm that had its own restraints. It was really fun learning a new side to game development and I enjoyed it a lot. This was one of the reasons I wanted to get into tool design/development.

polyBrush (Clone) Tool

This poly brush clone was the first tool I developed for Unity. This was my way of learning Unity's tool system which let me develop my skills. The circle reacts live to the terrain thanks to some cool math I used to make a lot of small lines that calculate the height and normal of a model. This live editor tool also allows you to snap to a grid with any object/prefab/model at the same time.

Sinking Sailors

Sinking Sailors was made for the 2020 global game jam, the theme was "repair". We won 2nd place awarded by CHCH Game Dev, a branch of NZGDA. The core gameplay is to stay afloat by fixing things with your teammates and to make it home by steering the ship away from rocks and sharks... good luck! I created the design, level design, and programmed core things like the ship sinking, ship movement, and the objects that needed to be dodged by the ship. I also worked on small things like particles when things break or get repaired.

Wingmans Halloween

You must host your butt off and get people to talk to each other. Give them food and drinks to help them find that right someone to make a vibing party. How to make friends

How to make friends is a project me and a friend made for the NZ game jam, the theme was "chain reaction". We both wanted to make something new and in a different engine to what we had made before while also tackling new tasks. So, my friend worked on the players while I programmed all the A.I. In this case, we have a combat system that both the AI and players shared, and 3 different A.I types. One is the standard grunt one, another was a bigger slower heavier hitting unit, and the last was fast and skinny. The summary is that it's a local multiplayer game where each player must build an army to take over the flag, like the king of the hill, with the players giving orders like "get more people for the army", "come to me", or "go attack that location". This trailer was edited by Min-Su Lee, check him out here : https://minwater2.github.io/
